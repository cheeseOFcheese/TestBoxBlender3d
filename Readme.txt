# Тестовое задание: Создание текстурированной коробки с открывающейся верхней крышкой

## Описание задания:
В ходе выполнения тестового задания была создана 3D-модель коробки с толстыми стенками и открывающейся верхней крышкой. Крышка открывается, функционируя как дверка или лифлет, без полного снятия.

## Характеристики коробки:
- **Горизонтальные размеры:** 160x160 мм
- **Высота:** Установлена по усмотрению исполнителя

## Текстурирование:
### Внешняя сторона коробки:
- Использовано изображение с цветочным узором, покрывающим всю внешнюю поверхность коробки.

### Внутренняя часть коробки:
- Использован однотонный цвет (или текстура однотонной шелковой ткани без складок) для внутренней части коробки.

### Внутренняя часть крышки:
- Использован второй однотонный цвет (или текстура однотонной шелковой ткани без складок) для внутренней части крышки.

## Формат и отправка:
Важно: Файл сделан в Blender 4.0
Все необходимые файлы упакованы в архив формата .zip. В архиве находится файл .blend с моделью, а также отдельные файлы с использованными текстурами. Текстуры могут быть предоставлены в формате jpg с разрешением 1024x1024.

## Примечание:
Это тестовое задание, и созданная модель не предназначена для использования в реальных проектах. Использовались текстуры небольшого разрешения и размера файла, чтобы минимизировать объем данных. Ничего дополнительно не покупалось в процессе выполнения задания.
